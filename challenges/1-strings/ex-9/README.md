# Number to String Function

## Objective:

Your task is to write a TypeScript function named `numberToString` that takes a number as its parameter and returns its string representation. For example, if you pass the number `123` to the function, it should return the string `"123"`.

## Function signature

```typescript
export function numberToString(num: number): string;
```

## Instructions:

1. **Create a file:** Create a file named `numberToString.ts`.
2. **Define the function:** Using the provided signature, implement the function so that it converts the given number to its string representation.
3. **Export the Function:** Make sure to export your function from the module so that it can be imported and tested in other places.
4. **Testing your function:** To verify the correctness of your function, run the following command in your terminal:

```Bash
npm test -- numberToString
```
